InboundXML

REST API

Before using any of the TelAPI REST methods available, you must authenticate your account. Do so by including your REST API credentials in the following way:

from telapi import rest
 
account_sid = 'AC6ed337e6790d47289c3d2073d3c6d0e4'
auth_token  = 'c6279c1dd08349e5829a8dfaf3642571'
client      = rest.Client(account_sid, auth_token)
account     = client.accounts[client.account_sid]

Notice the code above also instantiates and assigns an account object. Every method used throughout this helper library will use this main account class.

Interrupt Live Call

Documentation

To interrupt a call, simply fetch it and specify the redirect url:

call  = account.calls['{CallSid}']
 
call.url = 'http://foo.com/redirect.xml'
 
call.save()

Hangup Call

Documentation

To Hangup a call, simply fetch it and change the status:

call  = account.calls['{CallSid}']
 
call.status = 'completed'
 
call.save()
 
print "Call Status: %s" % call.status

Send Digits to Call

Documentation

To send digits to a call, the play_dtfm property is provide:

call  = account.calls['{CallSid}']
 
call.play_dtmf = '1234'
 
call.save()

To specify a particular leg (caller or answerer), use the play_dtmf_leg property:

call.play_dtmf_leg = 'aleg'

Play Audio to Call

Documentation

To play audio to a call, fetch it and then use the play methods provided:

call  = account.calls['{CallSid}']

play = call.plays.new()

play.audio_url = '{audio-file}'

play.save()

Record Call

Documentation

To record a call, fetch it and then use the recording methods provided:

call  = account.calls['{CallSid}']

r = call.recordings.new()

r.record = 'true'

r.callback_url = 'http://www.foo.com'

r.save()
